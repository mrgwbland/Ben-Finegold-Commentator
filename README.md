finegoldlichess
===========

finegoldlichess is a Google Chrome extension based off [dmitlichess](https://chromewebstore.google.com/detail/dmitlichess/haigaafonckooggfpmjplfpbkafelcfe?hl=en).
It adds commentary of Ben Finegold to your Lichess games, it includes a soundboard and an options page to control the speech rate.
If this is popular I could add more commentators, I'm also more than welcome to recieve suggestions for more audio clips I could add to Ben Finegold.
